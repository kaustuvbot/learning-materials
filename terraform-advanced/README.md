## How to Use This Course

Each module folder contains:
- **Numbered `.md` files** — one per subtopic, work through them in order
- **`examples/`** — complete, runnable Terraform configurations
- **`lab/starter/`** — incomplete code for hands-on practice
- **`lab/solution/`** — complete working solution
